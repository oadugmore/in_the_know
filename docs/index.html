Privacy

We, the developers of In The Know (“the app”), do not store or sell user data. When you use the app to search your location or some other query, this query is processed, and the results are displayed in the app. The app does not save the query or results. The query is passed through the Twitter API and is subject to Twitter’s privacy policy.
